# fast-workshop-1.0

Minimal Bangla e-commerce landing page for Fast Workshop's Fast Mobil Chula.

## Files

- `index.html` - GitHub Pages-ready website
- `style.css` - mobile-first responsive styling
- `script.js` - order form, product selection, payment fields, Google Apps Script submission
- `google-apps-script.gs` - Google Apps Script order receiver
- `product photo/` - product images used on the website

## Setup

1. Deploy `google-apps-script.gs` as a Google Apps Script Web App.
2. Update `scriptURL` in `script.js` with the deployed Web App URL.
3. Enable GitHub Pages from the repository settings using the `main` branch.
